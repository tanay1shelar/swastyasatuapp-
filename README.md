# HMCMS - Medical Camp Management System

## Folder Structure
- /assets: Contains all CSS, JavaScript, and image resources.
- /includes: Contains reusable UI components like headers, sidebars, and footers.
- /config: Database and configuration settings.
- /modules: The core application logic, divided into feature-based modules (e.g., dashboard, centers, reports).
- /api: API endpoints for AJAX calls.
- /uploads: User-uploaded files (images, documents).

## Setup
Point your local server (XAMPP/WAMP) to this directory. Loading the root URL will automatically redirect to the superadmin dashboard.
